This project is an online calendar, where you can follow the new issues of all your favorite comics.

It uses the [ComicVine](http://www.comicvine.com) [API](http://www.comicvine.com/api/) to retrieve information about the new issues.

You can see it in action [here](http://comics.remyg.fr).
